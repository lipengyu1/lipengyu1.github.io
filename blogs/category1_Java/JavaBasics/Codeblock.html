    lpy 12/18/2021 代码块

    # 代码块

    代码块又称初始化块，属于类中的成员，类似于方法，将逻辑语句封装在方法体中，通过{}包围起来。
    无方法名、参数、返回，只有方法体，而且不用通过对象或类显示调用，而是加载类是，或创建对象时隐式调用。

    # 基本语法

    [修饰符]{
        代码
    }；

    # 说明注意

    1. 修饰符可选，只能写static
    2. 代码块分为两类，静态代码块（static）和普通代码块
    3. 逻辑语句可以为任何逻辑语句
    4. ；号可以加，也可省略 相当于另一种形式的构造器（对构造器的补充机制），也可做初始化操作
      如果多个构造器中有重复语句，可以抽取到初始化块中，提高代码重用性

    # 使用细节

    1. static代码块也叫静态代码块，作用就是对类进行初始化，随着类的加载而执行，并且只会执行一次。如果是普通代码块，每创建一个对象，就执行
    2. 类什么时候被加载
        1）创建对象实例时
        2）创建子类对象实例，父类也会被加载
        3）使用类的静态成员时（静态属性，静态方法）
      A类 extends B类的静态块
    3. 普通的代码块，在创建对象实例时，会被隐式的调用 被创建一次，就会调用一次 如果只是使用类的静态成员时，普通代码块不会执行 小结：static代码块是类加载时执行的，只会执行一次。 普通代码块是创建对象时调用的，创建一次调用一次
    4. 创建一个对象时，在一个类 调用顺序是
        1）调用静态代码块和静态属性初始化（静态代码块和静态属性初始化调用的优先级一样，如果有多个静态代码块和静态变量初始化，则按他们定义的顺序调用）
        2）调用普通代码块和普通属性初始化（优先级相同，多个按顺序调用）
        3）调用构造器
    5. 构造器的最前隐含了super（）和普通代码块
    6. 创建一个子类对象时，他们的静态代码块，静态属性初始化，普通代码块，普通代码初始化，构造方法的调用顺序如下
        1）父类的静态代码块和静态属性（优先级相同，顺序执行）
        2）子类的静态代码块和静态属性（优先级一样，按顺序执行）
        3）父类的普通代码块和普通属性初始化（优先级一样，按顺序执行）
        4）父类的构造方法
        5）子类的普通代码块和普通属性初始化（优先级一样，按顺序执行）
        6）子类的构造方法
    7. 静态代码块只能直接调用静态成员（静态属性和静态方法），普通代码块可以调用任意成员

class Movie{
    private String name;
    private double price;
    private String directer;
    //三个构造器重载
    //1.三个构造器都有相同语句
    //2.代码冗余
    //3.可以将相同的语句放入到代码块中
    //4.无论调用那个构造器创建对象，都会先调用代码块的内容
    //5.代码块调用的顺序优先于构造器
    {
        System.out.println("一");
        System.out.println("二");
        System.out.println("三");
    }
    public Movie(String name) {
//        System.out.println("一");
//        System.out.println("二");
//        System.out.println("三");
        System.out.println("Movie(String name)被调用");
        this.name = name;
    }
    public Movie(String name, double price) {
//        System.out.println("一");
//        System.out.println("二");
//        System.out.println("三");
        this.name = name;
        this.price = price;
    }

    public Movie(String name, double price, String directer) {
//        System.out.println("一");
//        System.out.println("二");
//        System.out.println("三");
        System.out.println("Movie(String name,double price, String directer)被调用");

        this.name = name;
        this.price = price;
        this.directer = directer;
    }
}

    public class CodeBlockDetail01 {
    public static void main(String[] args) {

        //类被加载情况举例
        //1.创建对象实例时
//        AA aa = new AA();
        //2.创建子类对象实例，父类也会被加载,而且父类先被加载
        AA aa1 = new AA();
        //3.使用类的静态成员时（静态属性，静态方法）
        System.out.println(Cat.n1);

        //new 两个对象实例， static代码块是在类加载时执行的，由于DD类加载了一次，static代码块只会执行一次，
        // 普通代码块加载两次

        DD dd = new DD();
        DD dd1 = new DD();

        //如果只是使用类的静态成员，普通代码块不会执行
        System.out.println(EE.n1);
    }
}

    }
}

    }

    public class CodeBlockDetail02 {
    public static void main(String[] args) {
        A a = new A();//输出顺序：getN1  ，  A静态代码块（顺序调用）  ，getN2被调用 ， 普通代码块，构造器
    }
}

class A {
    public A(){
        System.out.println("构造器被调用");
    }
    private int n2 = getN2();
    //静态属性的初始化
    private static int n1 = getN1();
    //普通代码块
    {
        System.out.println("普通代码块");
    }
    //静态代码块
    static {
        System.out.println("A的静态代码块");
    }

    public static int getN1() {
        System.out.println("getN1被调用");
        return 100;
    }

    //普通方法（非静态方法）
    public int getN2() {
        System.out.println("getN2被调用");
        return 200;
    }

}

class AAA{
    static {
        System.out.println("AAA静态代码块");
    }
    {
        System.out.println("AAA普通代码块");
    }
    public AAA() {
        //(1)super();
        //(2)调用普通代码块
        System.out.println("AAA构造器被调用");
    }
}
class BBB extends AAA{

        System.out.println("BBB静态代码块");
    }
    {
        System.out.println("BBB的普通代码块");
    }
    public BBB() {
        //(1)super();
        //(2)调用普通代码块
        System.out.println("BBB构造器被调用");
    }
}
