    # 成员内部类

    成员内部类定义在外部类的成员位置，并且没有static修饰

    1. 可以直接访问外部类的所有成员，包含私有的
    2. 可以添加任意访问修饰符（pulic,protected,默认，private），因为它的地位就是一个成员
    3. 作用域和外部类的其他成员一样，为整个类体。
    4. 成员内部类->访问 ->外部类 ，访问方式：直接访问
    5. 外部类-> 访问-> 内部类，访问方式：创建对象，再访问
    6. 外部其他类 ->访问-> 成员内部类（三种）
      相当于吧new Inner08（）当做是outer06的成员
    Outer06.Inner08 inner08 = outer06.new Inner08();//outer06.new Inner08();

    在外部类中编写一个方法可以返回Inner08对象

    public Inner08 getInner08Instance() {
    return new Inner08();
}

    1. 外部类与内部类成员重名时，内部类访问的话，默认遵守就近原则，如果想访问外部类成员，则可以用（外部类名.this）访问
    public class MemberInnerClass {
    public static void main(String[] args) {
        Outer06 outer06 = new Outer06();
        outer06.t1();
        //外部其他类使用成员内部类的三种方式
        //第一种
        //相当于吧new Inner08（）当做是outer06的成员
        Outer06.Inner08 inner08 = outer06.new Inner08();//outer06.new Inner08();
        //第二种 在外部类中编写一个方法可以返回Inner08对象
        Outer06.Inner08 inner08Instance = outer06.getInner08Instance();
        inner08Instance.say();
    }
}

class Outer06 {
    private int n1 = 10;
    public String name = "一";

    public void hi() {
        System.out.println("hi方法");
    }

    //1.定义在成员内部类是定义在外部类的成员位置上
    //2.可以添加任意访问修饰符（pulic,protected,默认，private），因为它的地位就是一个成员
    public class Inner08 {//成员内部类
        private int n1 = 99999;
        public void say() {
            //可以直接访问外部类的所有成员，包含私有的
            //7）外部类与内部类成员重名时，内部类访问的话，默认遵守就近原则，如果想访问外部类成员，则可以用（外部类名.this）访问
            System.out.println("n1  " + "name= " + name + n1+this.n1);
            hi();
        }
    }

    //方法
    public void t1() {
        //使用成员内部类
        // * 5）外部类 访问 内部类 访问方式：创建对象，再访问
        Inner08 inner08 = new Inner08();
        inner08.say();
    }

    public Inner08 getInner08Instance() {
        return new Inner08();
    }
}
