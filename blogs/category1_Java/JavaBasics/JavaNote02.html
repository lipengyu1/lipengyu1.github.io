    # 标识符规则

    硬性要求

    1. Java中的标识符是由数组，字母，下划线,$,组成的,不能使用@,%,*等其他特殊的字符，不能以数字开头。
    2. 标识符不能使用Java关键字
    3. 标识符严格区分大小写
    4. 标识符的命名最好能反映出其作用，见名知意
      软性要求
    5. 类名：大驼峰，首字母大写，后面每个单词首字母大写
    6. 变量名：小驼峰，首字母小写，后面每个单词首字母大写
    7. 方法名：动宾结合的小驼峰
    8. 常量名：全部大写，单词之间用_间隔
    public class test02{
    public static void main (String[] args){
        int studentAge = 22;
        System.out.println(studentAge);
        studentAge = 23;
        System.out.println(studentAge);
    }

    1. 数值型：整型类型-byte（1字节），short（2字节），int（4字节），long（8字节）

    # 强制类型转换

    自动类型转换：目标类型大于源类型，如 double 类型长度为 8 字节， int 类型为 4 字节，double 类型的变量里直接可以存放 int 类型的数据。

    public class Test08{
    public static void main (String[] args) {
        long l1 = 2000;
        double d = 123.4;
        int n1 = (int)d;
        System.out.println(n1);//123
        int n2 = 4660;
        //高位数字丢失（高<-低）
        byte n3 = (byte)n2;//int 4字节，byte 1字节
        System.out.println(n3);//52
        int n4 = 15;
        byte n5 = (byte)n4;
        System.out.println(n5);//15
    }

    # 表达式与运算符

    运算符：一种“功能”符号，用以通知 Java 进行相关的运算。譬如：age 的值设置为 20 ，这时候就需要一个“=”，告诉程序需要进行赋值操作。

    public class OperatorDemo {
    public static void main(String[] args) {
        //存放两个操作数。存放结果
        int num1 = 5,num2 = 2,result;
        //加法 赋值运算符优先级低
        result = num1 + num2;
        //+在字符串进行运算时起拼接作用
        System.out.println("num1"+"+"+"num2"+"="+result);
        //减法
        result = num1 - num2;
        System.out.println(result);
        //乘法
        result = num1*num2;
        System.out.println(result);
        //除法
        num2 = 13;
        result = num2/num1;
        System.out.println(result);//2
        //取余
        result = num2%num1;
        System.out.println(result);//3
       //双目运算符，当一个操作数是小数时，结果为小数；两个操作数为整数时，结果为整数
       System.out.println("12.0/5="+12.0/5);//2.4
       System.out.println("12.0%5="+12.0%5);//2.0
       System.out.println("12.0+5="+(12.0+5));//17.0
       System.out.println("12*5="+12*5);//60
       System.out.println("12*5.0="+12*5.0);//60.0
    }
}

    # Java运算符

    ++n,--n都是对变量自身进行一个+1 或 -1操作，自增自减运算符放在变量的左边或右边；
    ++n先对变量进行+1,后再参与计算、打印、输出。
    n++先参与计算、打印、输出,后对变量进行+1。

    public class Test_ {
    public static void main(String[] args) {
        int m = 5,n = 6;
        int x = (m++)+n;// x = 5+6 =11,m = 5+1 = 6;
        int y = (--m)+n;// m = 6-1 = 5, y = 5+6 =11;
        System.out.println(x);//11
        System.out.println(y);//11
        int a =2;
        System.out.println(a++);//输出 a = 2，再进行自加，a =2 + 1 = 3，
        System.out.println(a);//3
        System.out.println(++a);//先进行自加 a= 3 + 1 = 4，再进行输出a = 4
    }
}

    public class Demo02 {
    public static void main(String[] args) {
        System.out.println("输入单词:");
        Scanner scanner = new Scanner(System.in);
        //next() 获取键盘的字符串
        String week = scanner.next();
        //将字符串中的字符全部转换为大写
        week = week.toUpperCase();
        switch (week){
            case "MONDAY":
                System.out.println("星期一");
                break;
            case "TUESDAY":
                System.out.println("星期二");
                break;
            case "WEDNESDAY":
                System.out.println("星期三");
                break;
            case "THURSDAY":
                System.out.println("星期四");
                break;
            case "FRIDAY":
                System.out.println("星期五");
                break;
            case "SATURDAY":
                System.out.println("星期六");
                break;
            case "SUNDAY":
                System.out.println("星期日");
