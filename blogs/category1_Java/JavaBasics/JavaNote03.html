    # 包

    • 在Java中，包主要有以下用途
      1. 包允许将类组合成较小的单元
      2. 有助于避免命名冲突
      3. 包允许在更广的范围内保护类、数据和方法
    • 包可以是类、接口和子包的集合
    • 包名的命名规范
      1. 通常都是用小写英文字母
      2. 一般为公司网址的反向排列，如： com.company.region.package

    # 包的使用

    • 创建包
      package mypackage;
    • 访问包中的类
      mypackage.My_Class
    • 导入包
      import java.util.*;
    • 导入包中的类
      import java.util.Arrays;
    • 导入常量
      import static java.lang.Math.PI;

    # 局部变量与成员变量

    • 局部变量与成员变量的区别
      1. 位置不同：成员变量位于类中，局部变量位于成员方法中
      2. 作用域不同：成员变量对整个类有效，成员方法可以访问到；局部变量只对当前方法有效
      3. 初始值不同：成员变量有默认值（int 0，double 0.0，String null）；局部变量无初始值，必须先赋值再使用
      4. 内存存储位置不同：成员变量位于堆内存中，局部变量位于栈内存
      5. 优先级不同：当成员变量与局部变量同名时，遵循就近原则，访问方法中的局部

    # 构造方法

    构造方法分为有参构造和无参构造，有参构造负责给对象成员属性赋值，构造方法无返回值,方法名与类名一致

    形参变量为方法中的局部变量，当局部变量与成员变量名冲突时，局部变量名有效。this关键字表示当前的对象（谁调用，谁就是当前对象）为了避免我们在构造方法中对成员变遣赋值出错的时候，可以使用this关键字。避免成员属性与局部变量同名的冲突;根据我们的变量的就近原则，会将传入的值赋给我们形参变量,无法完成方法。

    # static

    static修饰的成员变量为静态成员，也称为类成员。类无论实例化出多少对象，都会共用一块静态空间。

    # 普通成员变量与静态成员变量的区别

    生命周期不同

    • 普通成员变量：伴随的对象的产生而产生，伴随着对象的销毁而销毁
    • 静态成员变量：伴随着类第一次加载而产生，类销毁时类成员也会销毁

    # static修饰方法

    静态方法：不能访问普通成员变量，只能访问静态成员变量

    静态方法：只能访问静态方法

    强制在静态方法中访问普通成员方法，new出实例对象，通过对象去调用它自己的成员方法

    # 封装

    1. 将成员变量私有化
    2. 设置set/get方法后去成员属性值，设置成员属性值
    3. 在set/get方法中进行属性正确性校验
    package test.cop;

public class Student {
    private String name;
    private int age;
    private String sex;

    public String getName() {
        return name;
    }

    public void setName(String name) {
        this.name = name;
    }

    public int getAge() {
        //不要将错误值进行输出
        if (this.age <= 0) {
            age = 20;
        }
        return age;
    }

    public void setAge(int age) {
        //判断Age的正确性
        if (age <= 0) {
            System.out.println("输入的年龄有误，系统将设置为默认值");
            age = 20;
        }
        this.age = age;
    }
    public String getSex() {
        return sex;

    public void setSex(String sex) {

            System.out.println("输入的性别有误，系统将设置为默认值");
            sex = "男";
        }
        this.sex = sex;
    }

    public void sayInfo(){
        System.out.println("我是名叫"+name+",年龄为"+age+"岁的"+sex+"生");
    }
}

    # 重载与重写

    • 方法的重载：根据方法调用传入的参数类型，来确定调用哪个方法

      1. 成员方法出现在一个类中
      2. 方法名相同，参数列表不同 构造方法（有参、无参）

    • 方法重写、覆盖：子类对继承的方法不满意的时候，可将继承下来的方法重写

      1. 出现在继承的子类中，子类对父类的方法不满意
      2. 子类中的方法名、参数必须与父类的方法完全一致

    # super

    子类中使用super. 可以调用父类的属性与方法

    子类中使用super（）可以调用父类的无参构造，不添加则默认调用父类的无参构造

    子类中使用super（参数列表）可以调用父类的有参构造

    • 例题1
    package test.extends02;
public class Aniaml {
    String name;
    String species;
    int age;
    public void eat(){
        System.out.println("动物在吃");
    }
    public Aniaml() {
        System.out.println("调用Animal的无参构造");
    }
    public Aniaml(String name, String species, int age) {
        this.name = name;
        this.species = species;
        this.age = age;
        System.out.println("调用Animal的有参构造");
    }
}

package test.extends02;
public class Dog extends Aniaml{
    /**
     * 1.先调用父类的构造方法，在调用子类构造方法
     * 2.子类会默认调用父类的无参构造super(),super()必须写在第一行
     * 3.子类有参构造显示调用父类的有参构造，父类成员属性才能被赋值
     */
    String sex;
    @Override
    //子类中使用super. 可以调用父类的属性与方法
    public void eat() {
        super.eat();
        System.out.println("狗在吃骨头,调用父类特有的方法");
    }
    //子类中使用super（）可以调用父类的无参构造，不添加则默认调用父类的无参构造
    public Dog(){
        super();
        System.out.println("Dog的无参构造被调用");
    }
    //子类中使用super（参数列表）可以调用父类的无参构造,
    //能够将父类属性一并赋值,显示调用父类的有参构造
    public Dog(String sex) {
        super("哈哈","大狗",18);
        System.out.println("Dog的有参构造被调用");
        this.sex = sex;
    }
}

package test.extend_;
public class Child extends Father{
    //孩子的
    int money=0;

    public void showMoney(){
        System.out.println("爸爸的存款"+super.money);
        System.out.println("儿子的存款"+this.money);
    }
    public void goSchoolPerYear(){
        System.out.println("-----------------------");
        System.out.println("开学了，交学费");
        super.money-=20000;
        System.out.println("-----------------------");
    }
    public void goJobPerMonth(){
        System.out.println("--------搬砖生活开始，坐等发工资----------");
        this.money+=10000;
    }
    public void getMarry(){
        System.out.println("Marry");
        if (this.money<300000){
            this.money = 0;
            super.money -= (300000 -this.money);
        }else{
            this.money -= 300000;
        }
        System.out.println();
    }
}

package test.ploy;
public class Test {
    public static void main(String[] args) {
        //多态：运行时多态，程序运行时会根据对象的类型动态调用对象的方法
        //animal对象执行相同的方法会有不同的相应，运行过程中父类的对象会动态绑定到子类对象上，执行子类的方法

        Animal animal = new Animal();
        animal.eat();//动物在吃

        //父类的引用指向子类的对象，相同父类的引用指向不同类型子类实例对象，
        //虽然方法名相同  会恒根据不同的对象用不同的调用
        Animal animal1 = new Cat();
        animal1.eat();//小猫在吃东西
//        animal1.run();错误，向上转型不能调用子类特有的成员，只能调用父类所有方法及子类重写的方法
        Animal animal2 = new Dog();
        animal2.eat();//狗吃骨头
//        animal2.watch();错误，向上转型不能调用子类特有的成员，只能调用父类所有方法及子类重写的方法


        //转成子类，向下转型
        //子类引用是否能指向父类对象，必须进行强转
        if (animal1 instanceof Cat){
            System.out.println("animal2可以转换为Dog类型");
            Cat cat1 = (Cat)animal1;
            cat1.run();
        }else {
            System.out.println("animal2不可以转换为Dog类型");
        }

        //animal具体指向猫对象，猫无法转成狗
        //向下转型时，一定要关注父类引用指向的对象类型，才能向对应的子类类型进行转换，提前加判断
        //转型后，可以调用子类特有的成员
        if (animal2 instanceof Dog){
            System.out.println("animal2可以转换为Dog类型");
            Dog dog1 = (Dog)animal2;
            dog1.watch();
        }else{
            System.out.println("animal2不可以转换为Dog类型");
        }
    }

    Equals 是Object中的方法，只能判断引用类型,默认判断的是地址是否相等;子类中往往重写该方法，用于判断内容是否相等。
    若某个类没有覆盖equals()方法，当它的通过equals()比较两个对象时，实际上是比较两个对象是不是同一个对象。这时，等价于通过“==”去比较这两个对象。
    我们可以覆盖类的equals()方法，来让equals()通过其它方式比较两个对象是否相等。通常的做法是：若两个对象的内容相等，则equals()方法返回true；否则，返回fasle。

        public boolean equals(Object anotherPhone){
        /**
         * 1.判断两个对象地址相同 return true
         * 2.类型判断，两个对象类型不同 return false
         * 3.两个对象类相同，地址不同，对成员属性进行一一比较（字符串equals 比较内容，整型、浮点型 ==）
         */
        if (this == anotherPhone)
            return true;
        if (anotherPhone instanceof Phone){
            Phone newPhone = (Phone) anotherPhone;
            return this.name.equals(newPhone.name)
                    &&this.size == newPhone.size
                    &&this.price == newPhone.price;

        }
        return false;
    }
